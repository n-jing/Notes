  • win 路径默认不分区大小写，与Linux不同，可以开启，请参考。
  • win 默认开启ssh client服务，需手动开启ssh server服务，首先需要安装OpenSSH Server，然后启动该服务，请参考

Explorer.exe 是用户与电脑里文件系统、文件互动的软件，类似 Mac OS 里的 Finder，DOS 时代的 http://command.com，各种UNIX liker 下的各种 Shell。

添加病毒扫描排除项， win+r, powershell -inputformat none -outputformat none -NonInteractive -Command Add-MpPreference -ExclusionPath $ENV:USERPROFILE\Downloads, ctrl+shift+{Enter}

win 默认以普通用户权限执行任务，即使是用管理员账户登录；当需要提升权限时，系统会给出提示。请参考

win 终端代理设置

请参考
trojian只提供了一个端口，如果command prompt需要代理，则还需要vpn软件，这里使用v2ray+v2rayN，请参考

  1. 下载v2ray和v2rayN，解压到同一个目录。
  2. 添加socks服务，socks默认端口10808，http默认端口10809。
  3. 配置代理规则，将需要代理的网址添加到“设置”-“路由设置”-“代理的domain”，请参考

如果IE需要代理，则在“Internet属性”-“连接”-“局域网设置”-“代理服务器”中设置。
